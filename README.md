## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## !

To execute the Docker image hosted in Github, you need to provide Github credentials to the Docker daemon. This can be achieved executing
```
docker login docker.pkg.github.com
docker-compose up -d
```
